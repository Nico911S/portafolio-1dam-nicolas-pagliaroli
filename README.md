# Portfolio 1º DAM - Nicolás Pagliaroli

Este repositorio contiene trabajos, prácticas y ejercicios realizados durante el primer curso de DAM, principalmente en las asignaturas de **Lenguaje de Marcas** y **Programación**.

El objetivo de este portfolio es presentar de forma ordenada el trabajo realizado durante el curso.

---

## Lenguaje de Marcas

En la carpeta **Lenguaje de marcas** se incluyen prácticas relacionadas con el desarrollo web.

Contenido incluido:

- Prácticas de HTML
- Prácticas de CSS
- Bootstrap
- Estructura de páginas web
- Imágenes y recursos
- Ejercicios y prácticas realizadas durante el curso

Conceptos trabajados:

- Estructura HTML
- Etiquetas semánticas
- Uso de `header`, `nav`, `main`, `section`, `article`, `div` y `footer`
- CSS externo
- Flexbox
- Diseño responsive
- Bootstrap
- Organización visual mediante secciones y tarjetas

---

## Programación

En la carpeta **Programacion** se incluyen ejercicios y proyectos realizados en Java durante el curso.

Contenido incluido:

- Ejercicios realizados con IntelliJ
- Ejercicios realizados con Notepad
- Prácticas por unidades
- Código fuente en carpetas `src`
- Ejercicios de Programación Orientada a Objetos

Conceptos trabajados:

- Clases y objetos
- Encapsulamiento
- Herencia
- Métodos
- Atributos
- Constructores
- Arrays
- ArrayList
- HashMap
- HashSet
- Enumerados
- Excepciones
- Menús interactivos
- Validaciones
